# corporate-advisor — 合同会社多元現実 税務・法務・労務 自己完結リファレンス

税理士・弁護士・社労士への相談を代替する、社内の一次知識ベース。何か会社運営の判断に当たったら、外部専門家に振らずまずここを読み、公式一次ソースを引いて結論まで出す。

## 運用原則

- **結論を出す。**「専門家に確認してください」で終わらない。グレーな点は「保守的にこうする」と既定値を示す。
- 根拠は必ず**公式一次ソース**(国税庁/e-Gov/日本年金機構/厚労省)のURLを示す。税理士ブログ等は補助に留め、canonicalにしない。
- 判断が割れる論点は**保守的実務解(=税務調査で否認リスクが最小の選択)**を明示して選ぶ。
- 数値・式は「キリのいい概算」で言わず、**採用する計算式を1本固定し、その式の出力をそのまま使う**。入力値(在宅日数・床面積等)を月次で記録に残す。
- 改正・判例で事実が変わったら、該当章末の `[要更新: YYYY-MM]` を更新する。

## Part 1: 税務

### 1-1. 役員社宅 — 賃貸料相当額と給与課税回避

法人が物件を借り上げ役員本人に貸与する場合、役員から徴収する家賃が「賃貸料相当額」を下回ると、差額が現物給与として役員報酬(定期同額給与)に加算され源泉徴収義務が生じる。

**物件分類(当物件):** RC1983築 → 耐用年数47年 > 30年 → 「30年超の建物」区分。30年超の小規模住宅は床面積99㎡以下。26㎡ → **小規模住宅に該当**。

**賃貸料相当額/月(小規模住宅):**
```
建物固定資産税課税標準額 × 0.2%
+ 12円 × (総床面積㎡ ÷ 3.3)
+ 敷地固定資産税課税標準額 × 0.22%
```
固定資産税課税標準額は固定資産税評価証明書または家主/管理会社から取得。

**実務:** 計算した賃貸料相当額以上を毎月役員→法人へ口座振込(証拠最強)。差額ゼロなら給与課税なし。

### 1-2. 通信費(home 5G)の按分と私用分処理 ★本件確定解

> ⚠️ **2026-06 補足(要整合)**: ①当社は当期**免税事業者**(税込経理)なので、本節の仮払消費税(課税仕入)計上は**現状は当てはまらない**(税込でそのまま・仕入税額控除なし)。本節は将来課税事業者になった場合の型として読む。②home 5G は**まだ未購入**(2026-06 本人確認)。下記は契約時の設計案。③**按分の法的建て付けを補正(2026-06-24 一次ソース再検証)**: 通信費の会社負担は出発点では原則 経済的利益(給与)課税の論点(所基通36-26の光熱費等の非課税条件は寄宿舎・共同施設を想定したもので、賃貸マンション形式の借り上げ社宅には原則不適用)。業務分を損金化するには「業務利用が主体」の客観的立証(業務専用アカウントのみ登録／私用は別回線で分離／利用明細の保存)が要る。前提として**賃貸借契約書の用途記載(住居専用/事務所兼用)を確認**(住居専用だと業務按分の主張と用途が矛盾し税務調査リスク)。確定解の出どころ: [⏳社宅のインターネット導入（home 5G）] 2026/6/24 節。

**前提:** 法人名義 home 5G。月額は法人チャネル(ドコモビジネスオンラインショップ)で5,280円→月々サポート▲1,525で**実支払3,755円/月(48ヶ月、49ヶ月目以降5,280円)**。端末HR02=**73,260円(税込/net66,600円)を一括購入**(法人チャネルは分割でなく一括なので途中解約の端末残債なし)。社宅設置、業務+私用混在の定額回線。

**確定方針:** 業務按分後の業務分のみを通信費(課税仕入)計上、私用分は**役員からの実費入金(立替金精算)**で消す。雑収入・役員報酬控除・現物給与・100%/70%按分は使わない。

**既定按分率 = 業務35% / 私用65%。** 根拠は国税庁・在宅勤務FAQ(令和3年1月)の式に固定する:
```
業務分 = 実支払月額 × (在宅業務日数 ÷ 月日数) × 1/2
       = 3,755 × (22 ÷ 31) × 1/2 ≈ 1,314円(≈35%)   ※49ヶ月目以降は5,280で再計算
```
- 「×1/2」は在宅日数比に掛ける係数。1/2単独=50%ではない。50%/70%/100%は否認リスクが上がるため採らない。
- 按分は会社が**実際に支払う月額**(当面3,755円、49ヶ月目以降5,280円)に適用する。
- 事業拠点性(ミートアップ・デモ・来客)で35%を上回って計上したい場合のみ、来客簿/ミートアップ記録を1件ずつ日付・参加者・内容で証跡化した範囲で式の係数を引き上げ、社内文書で根拠化する。証跡なしの上乗せはしない。
- **端末73,260円(net66,600<10万円)は会社所有のルーター**。少額減価償却資産として**消耗品費で一括損金**。本体は所有(消費でない)ため原則100%会社資産として扱い、按分は毎月の通信料にのみ適用。最大限保守的にするなら本体も35%按分し私用分(65%≈47,619円)を役員から徴収。

**freee仕訳(毎月・既定35%・在宅22日/31日の例):**

| 区分 | 計算 | 金額 |
|---|---|---|
| 業務分(通信費・課税仕入10%適格) | 3,755 × 35% | 1,314円 |
| 私用分(立替金→役員回収・対象外) | 3,755 × 65% | 2,441円 |
| 合計(法人支払額/月) | — | 3,755円 |

```

              立替金 2,441 /                (私用分・対象外/不課税)
仕訳2 支払:   未払金 3,755 / 普通預金 3,755

```
- ドコモ請求書PDF(=適格請求書)を取引に添付。按分計算メモ1行「在宅◯日 ÷ 月◯日 × 1/2 = ◯%」を残す。
- **立替金が月末残高に残らないこと(回収漏れ=現物給与論点の再燃)を月次で確認。**役員入金は口座振込で証拠化。

**摘要テンプレ:**
```
home5G 2026年●月分 業務按分35% 在宅22日/31日×1/2 法人名義契約
home5G 2026年●月分 役員私用65% 立替金(役員実費精算で回収)
home5G 2026年●月分 役員私用分 実費精算入金(立替金消込)
```

**なぜ立替金方式か(否認レビュー反映):**
- 雑収入計上は法人の課税売上になり消費税区分が狂う → 立替金なら損益・課税売上を通さず消える。
- 役員報酬天引きは定期同額の改定タイミング(期首3か月)に縛られ期中導入で損金不算入リスク → 報酬に触れない。
- 仕入税額控除は業務分のみ。私用分は立替金で控除から除外され整合。

**社宅家賃との関係:** 別規定。社宅の賃貸料相当額計算式は通信費に適用しない。ただし同一26㎡の部屋を社宅ルールでは「居住用」、通信按分では「業務拠点」と扱うため、通信を高按分(70%等)にすると「居住用社宅」性に疑義が出て社宅家賃側の否認を誘発しうる。→ 物件は「居住用社宅 + 付随的業務利用」と位置づけを統一し、通信按分は控えめ(式準拠35%)に抑え両論点を矛盾させない。

**法人名義の理由:** 損金性立証が容易・インボイス受領で仕入控除可・在宅勤務FAQ前提に整合。ただし名義は必要条件であって十分条件ではない。事業供用証跡(来客簿/会議ログ)の継続記録が必須。

### 1-3. 役員報酬 — 定期同額給与

要件(法法34条1項1号): 支給時期1か月以下・各支給時期同額・変更は**事業年度開始から3か月以内**に社員の決定で行う。社宅按分や通信費現物給与を後から報酬に組み込むと期中変更=損金不算入リスク。**期初に現物給与込みの総額を決める**(本件は立替金方式で現物給与を発生させないので原則この論点を回避)。

### 1-7. 社宅備品の経費化(役員立替) ★2026-06 実施済

役員(代表)が個人決済で買った社宅の業務用備品を法人で損金化する型。**会社所有・役員へ貸与**を貫けば現物給与にならず損金算入できる(法基通9-2-10)。

- **損金化の条件**: ①業務関連性(使用目的・設置場所の記録) ②**会社所有**(備品台帳/freee摘要に明記、所有権は法人) ③役員へは**貸与**(返却義務あり・現物支給=所有権移転にしない) ④証憑(メルカリ/ジモティー取引画面・決済記録) ⑤**役員報酬勘定で経理しない**。
- **金額**: 1点10万円未満は消耗品費(`account_item_id=1051646037`)で即時損金(法税施令133条)。中古・新旧不問。
- **按分**: 居住兼用什器は業務割合で按分(例 在宅ワークチェア70%/充電器50%)。生活家具(ダイニング/スツール)は役員1人供与で所基通36-29の非課税例外に触れやすく**計上見送りが安全**。
- **消費税**: 当期免税なので税込でそのまま(インボイス無関係)。
- **記帳の型**: 証憑を `freee_file_upload` → 取引(消耗品費)を起票 → 送金(法人→個人) → **送金明細をその取引へ消込**。
- ★**罠(2026-06実証)**: 「振込 タカギシユンスケ」明細を**自動で経理ルールが役員報酬に誤登録**する(定期同額でない→損金不算入＋源泉)。**必ず既存の未決済取引へ消込**。誤登録したら当該取引を消耗品費へ作り替え(PUT)＋重複取引を削除で修正。
- 実例: 2026-06 家具6品¥80,100を freee 取引3606765435 に統合・決済済(証憑7点)。詳細→ plural-reality Scrapbox「☑️社宅の備品の経費化について調査」。

### 3-2. 労働保険(労災) — 労働者を1人でも雇ったら ★2026-06 確定解

労働者(パート・学生・短期含む)を1人でも使用した時点で労災の保険関係が**当然成立**(徴収法3条)。届出の遅れに関わらず雇用初日に遡って成立。役員のみ(労働者0)なら不要。

**手続き(遅延・遡及の是正もこの順):**
1. 保険関係成立届(**様式第1号**)→ 所轄**労基署**。本来は成立日から10日以内だが遅延提出可・成立日に遡って処理。一元適用事業は労基署のみ。
2. 概算保険料申告書(**様式第6号**)→ 成立から50日以内。翌年6/1〜7/10の年度更新で確定保険料申告。
3. 添付: 法人は**履歴事項全部証明書(登記事項証明書)**。本店移転後は新住所版。

**継続事業は事業廃止まで保険関係が消滅しない(徴収法5条)。** 労働者がゼロ/「当面雇わない」意思だけでは消滅届は出せない(会社清算・廃業時のみ)→ 成立届を出し翌年度更新で確定(確定賃金で精算・翌年度概算0円)。

**保険料:** 賃金総額の**1,000円未満切捨**(徴収法15条)× 労災率。「その他の各種事業」(番号94)の率=**3/1,000**(令和6年4月〜・令和8年度据置)。
- **追徴金(10%)**=政府の認定決定時のみ(徴収法21条)→**自主申告なら課されない**。
- **延滞金**=保険料額が**1,000円未満なら徴収しない**(徴収法27条)。最低保険料の規定なし。

**提出経路:** e-Gov(GビズIDプライムあれば電子証明書不要)。無ければ**紙様式を労基署へ郵送/持参が最短**(GビズIDプライム取得は印鑑証明+郵送で数日)。

**worked example(秋葉杏介・2026/4/1〜6/11・賃金総額¥25,400):** 25,400→切捨25,000 × 3/1000 = **保険料75円**。追徴・延滞なし。様式1号+6号+履歴(新住所版)を渋谷労基署へ→75円納付で是正完了。**電話確認不要**。

### 3-3. 短時間・学生・短期労働者の社保/雇用保険 適用判定 ★2026-06 確定解

- **社会保険(健保・厚年)**: 週所定が通常労働者の**概ね3/4以上**で適用。3/4未満は「特定適用事業所(従業員51人超)＋月8.8万円以上・週20h以上・学生でない 等」を全て満たす時のみ適用。**昼間学生は原則適用除外**。零細(被保険者1〜2名)は特定適用事業所でないので3/4基準のみ→短時間学生は不要。
- **雇用保険**: ①週所定**20時間以上** かつ ②**31日以上**の雇用見込み の両方で被保険者。**昼間学生は原則適用除外**。どちらか欠けば設置届・資格取得届ともに不要。
- 判定例(秋葉): 週20h未満相当・昼間学生 → 社保・雇用保険ともに**不要**(労災のみ必要)。

### 3-4. 給与所得の源泉徴収票(退職者交付) ★2026-06 確定解

年の途中で退職した者には、**退職後1か月以内**に「給与所得の源泉徴収票」を本人交付(所法226条)。乙欄者・年末調整未済でもそのまま交付。受給者交付用は個人番号欄を省略可。
